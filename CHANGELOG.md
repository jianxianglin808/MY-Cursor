## 版本说明

### 版本号规则
本项目遵循[语义化版本](https://semver.org/lang/zh-CN/)规范：

- **主版本号**：进行不兼容的 API 修改时
- **次版本号**：向下兼容的功能性新增时
- **修订号**：向下兼容的问题修正时

### 更新类型说明

- ✨ **新增**：新增功能
- 🐛 **修复**：Bug 修复
- ⚡ **优化**：性能优化或体验提升
- 💥 **破坏性变更**：不向下兼容的修改
- 📖 **文档**：文档相关修改
- 🔧 **配置**：配置文件相关修改
- 🎨 **界面**：界面样式调整
- 🔒 **安全**：安全相关修复
